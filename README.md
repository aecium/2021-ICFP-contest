# Free Tacos - ICFP Contest 2021

## Manifesto

Set the tacos free!

You will need to creat a .env in the tools dir with the var APIKEY=your teams api key

## WUI Usage
```
# Build the latest solvers
cargo build --release

# Start the web server
cd wui
cargo run
```

Open http://127.0.0.1:8000/ in a web browser.
